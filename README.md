# OpenPharma

Open-source EMA regulatory intelligence dashboard.

Free, MIT licensed. No vendor contracts. No paywalls.

---

## What it does

OpenPharma pulls publicly available data from the European 
Medicines Agency (EMA) and presents it in a clean, 
navigable dashboard — the intelligence layer that 
regulatory affairs teams, market access teams, and pharma 
SMEs currently pay €20K–€100K/year to access through 
gated platforms like IQVIA and Citeline.

**Data sources:**
- EMA drug approvals (EPAR database)
- EudraVigilance adverse event reports
- EMA drug shortage database
- EU Clinical Trials Register
- EU AI Act classification layer (August 2026)

---

## Who it is for

| Audience | Use case |
|---|---|
| Regulatory affairs teams | Track EMA approvals and EPAR summaries |
| Market access teams | Monitor drug shortages and approval timelines |
| Pharma SMEs and biotechs | Free alternative to Citeline and IQVIA |
| HTA bodies | Approval data for health technology assessment |
| Academic researchers | Open regulatory intelligence for EU medicines |

## Run locally

```bash
git clone https://github.com/lippershey-co/openpharma
cd openpharma
pip install -r requirements.txt
streamlit run app.py
```

Add your Anthropic API key to a `.env` file:


ANTHROPIC_API_KEY=your_key_here
